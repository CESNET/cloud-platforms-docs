## About PaaS Cloud

PaaS Cloud is e-INFRA CZ's managed container platform, built on [Cozystack](https://cozystack.io/). It delivers an opinionated and easy-to-use platform for Kubernetes and modern cloud services that complements our IaaS Cloud offering.

## When to Choose PaaS Instead of IaaS

| IaaS (OpenStack) | PaaS (Cozystack) |
|---|---|
| You need full control over VMs and network | You want to run containers and databases without managing servers |
| You want to manage your own Kubernetes cluster | You need Kubernetes as a Service with faster lifecycle management |
| You need custom images and nested virtualization | You want to deploy applications from the marketplace in a few clicks |

## Key Features

- **Kubernetes as a Service (K8saaS)**: Multi-tenant clusters with self-service scaling and zero node management.
- **Application Marketplace**: Deploy managed databases, registries, message queues, virtual machines, and more from the dashboard.
- **Managed Networking**: Automatic load balancing, ingress, and tenant network isolation.
- **Built-in Monitoring**: Grafana-based stack for dashboards, logs, and alerting.
